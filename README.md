# Retail Performance Analytics Project

## Project Overview
End-to-end retail sales analysis of 9,994 transactions 
across the United States using Excel, SQL and Power BI.

## Tools Used
- Microsoft Excel — Data cleaning and Pivot Tables
- SQL (SQLite) — Data querying and analysis
- Power BI — Interactive dashboard and visualization

## Key Business Insights
- West region leads sales with $725,457 (32% of revenue)
- Furniture has 2.5% profit margin despite high sales
- Texas is losing $25,729 — highest loss making state
- Canon Copier is top product at $61,599 in sales

## Files in this Repository
- Superstore.csv — Raw dataset
- RetailData_Cleaned.xlsx — Cleaned Excel file with Pivot Tables
- SQL_Queries.txt — All 5 SQL queries with results
- RetailDashboard_PDF.pdf — Power BI Dashboard export
- Retail_Insights_Report.pdf — Business insights report

Data Cleaning, SQL Querying, Data Visualization,
